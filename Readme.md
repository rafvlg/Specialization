__Задача__

Написать программу, которая из имеющегося массива строк формирует массив из строк, длинна которых меньше либо равна 3 символа. __Примеры__

["hello", "2", "word", ":-)"] ---> ["2", ":-)"]
["1234", "1567", "-2", "computer science"] ---> ["-2"]
["Russia", "Denmark", "Kazan", ""] ---> []
